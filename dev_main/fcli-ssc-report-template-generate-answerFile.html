Name

fcli-ssc-report-template-generate-answerFile - Generate a new answer file template that can be used for the creation of new report template definitions. For example, within the answer file, you can specify things like the name of the report definition, what parameters it accepts, and what are the types of those parameters.

Synopsis

fcli ssc report-template generate-answerFile [--force] [-f=<filePath>]

Description

Options

-f, <filePath>

Output file

--force

Overwrite otput file if it already exists
